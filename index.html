「Webコンテンツ作成」の成果

本テーマの学習目的

このテーマでは、HTML、CSS、JavaScriptの基本的な構造や記述方法を学び、Webページを制作する方法を勉強する。 HTMLでは文書を構造化する方法を、CSSでは構造化された文書を装飾する方法を学ぶ。 そして、JavaScriptではWebページの動的な処理やイベント処理を実装する。 以上3つの言語を使用して、具体的な書き方を調べながら様々な要素を組み合わせて簡単なWebページを作成できることが目的である。

プロジェクト課題について

プロジェクト課題では、3種類の内で「世界の天気」を選び、世界各都市の天気情報も持ってきて表示するウェブサイドを作成することにした。 今回の課題では、 都市名を指定して、「天気」「最高気温」「最低気温」「湿度」「風速」を表示するようにした。 また、本プロジェクト課題に検索欄に地区idを入力してその地区の天気情報が表示される。

プロジェクト課題で作成したページ:project.html

数当てゲーム

3回をかけて数当てゲームのページ。

乱数で 1〜10の値を1つ決めて,ユーザは答えを予想して,数を最大で3回入力できる. ユーザの入力が外れた場合は，入力よりも答えが大きいか小さいか表示する. 3回以内に答えの数を当てればユーザの勝ち.そうでなければ負け.

数当てゲームのページ: kazuate.html

第2回 CSSによる Web ページの装飾

今回はCSSの書き方について学んだ。CSSはHTMLで構造を指定した文書の装飾を行うファイルである。 CSSはセレクタとプロパティで構成される。 セレクタでは、そのプロパティで指定された装飾を行う要素やクラスを指定する。プロパティは、文字サイズや色、背景色、関数を書くことができる。

演習の成果

  • 演習2-3:ウェブブラウザ説明ページの装飾

第3回 JavaScript の基本

今回はJavaScriptについて学んだ。JavaScriptはプログラミング言語であり、ウェブ上で動的な処理を実現することができる。 HTMLのbodyタグ内にを追加して直接記述する方法と、headタグ内にを記述してファイルを指定する方法がある。 deferは実行を先送りする命令で、必ず書くこと。

今回はDOM_APIを学んだ。ブラウザのコンソール上でプログラムを実行して要素や要素などの「セレクター」を検索することができる。 このDOM_APIを使用して、HTMLの追加や削除、クラスの追加、CSSの設定変更など、下記の項目をJavaScriptファイルで実行できる。

第5回 イベント処理

今回はイベント処理を学んだ。イベント処理はJavaScriptに記述する。 [要素].addEventListener('イベント', [関数名「（）」は付けない]) のように書くことで、指定されたことが起きた時、上記の関数が呼び出される。
